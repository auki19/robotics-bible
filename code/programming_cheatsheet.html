FTC Java Cheatsheet

Return type Method Description

DC Motor (DCMotor)

void setPower(double power) Sets the motor's power (speed), within the interval [-1.0, 1.0]
void setDirection(DcMotor.Direction.FORWARD) Set motor direction to forward movement.
void setDirection(DcMotor.Direction.REVERSE) Set motor direction to reverse movement.
void setTargetPosition(int position) Sets a target position for the motor to run to.
void setMode(DcMotor.RunMode.RUN_TO_POSITION) The motor continues rotating at the speed specified by setPower() until it reaches the rotation amount specified by setTargetPosition().
void setMode(DcMotor.RunMode.STOP_AND_RESET_ENCODER) Stops motor movement and resets the current encoder position to 0.
void setMode(DcMotor.RunMode.RUN_WITHOUT_ENCODER) Tells the motor to run without an encoder (using continuous movement rather than a target position)
void setMode(DcMotor.RunMode.RUN_USING_ENCODER) Tells the motor to use its encoder (run to a target position rather than using continuous movement)
void setZeroPowerBehavior(DcMotor.ZeroPowerBehavior.BRAKE) When the motor is recieving zero power, it will brake and actively resist movement (i.e. can't be manually rotated). Generally you should use this ZeroPowerBehavior mode for motors.
void setZeroPowerBehavior(DcMotor.ZeroPowerBehavior.FLOAT) When the motor is recieving zero power, it will not move on its own, but won't actively resist movement (can be manually rotated).
boolean isBusy() Returns true if the motor is currently moving towards a target position.
int getCurrentPosition() Returns the current encoder reading.

Servo (Servo)

void setPosition(double pos) Sets the angular position of the servo.
void setDirection(Servo.Direction.FORWARD) Makes the servo rotate forward.
void setDirection(Servo.Direction.REVERSE) Reverses the direction the servo rotates in.

Misc. (Not called on a variable)

public void runOpMode() { } The main method that is called when INIT is selected on the Driver Station app.
void waitForStart() Pauses the program until the start button is selected on the Driver Station app, thereby separating the initialize and run phases.
boolean opModeIsActive() Returns a boolean which is true if the program is currently in its "run" phase and false otherwise.

Threads

Thread nameOfThread = new Thread() { } The statement you write to create a Thread, a chain of events independent from the main method or any other thread. This means a sleep statement in one Thread or the main method won't pause a different Thread, or vice versa.
public void run () { } The main method within a Thread (replaces what would be public void runOpMode() { } outside of the thread)
