FTC Java Cheatsheet

DC Motor (DCMotor)

void setPower(double power) Sets the motor's power (speed), within the interval [-1.0, 1.0]
void setDirection(DcMotor.Direction.FORWARD) Set motor direction to forward movement.
void setDirection(DcMotor.Direction.REVERSE) Set motor direction to reverse movement.
void setTargetPosition(int position) Sets a target position for the motor to run to.
void setMode(DcMotor.RunMode.RUN_TO_POSITION) The motor continues rotating at the speed specified by setPower() until it reaches the rotation amount specified by setTargetPosition().
void setMode(DcMotor.RunMode.STOP_AND_RESET_ENCODER) Stops motor movement and resets the current encoder position to 0.
Don't know what an encoder is? Check out the hardware cheatsheet.
void setMode(DcMotor.RunMode.RUN_WITHOUT_ENCODER) Tells the motor to run without an encoder (using continuous movement rather than a target position)
void setMode(DcMotor.RunMode.RUN_USING_ENCODER) Tells the motor to run with an encoder (run to a target position rather than using continuous movement)

Servo (Servo)

void setPosition(double pos) Sets the angular position of the servo.
void setDirection(Servo.Direction.FORWARD) Makes the servo rotate forward.
void setDirection(Servo.Direction.REVERSE) Reverses the direction the servo rotates in.
